# 訂單表單網頁

這是一個靜態的 GitHub Pages 網頁，提供訂單表單功能，允許用戶填寫訂購資訊並將資料儲存到 Google 表單。

## 功能

- 訂購者資訊輸入
- 聯絡電話輸入
- 訂購項目和數量選擇
- 希望交期選擇（日曆選擇模式）
- 照片上傳功能
- 資料自動儲存到 Google 表單

## 使用方法

1. 填寫表單中的必要資訊
2. 選擇訂購項目和數量
3. 使用日曆選擇希望交期
4. 上傳相關照片（可選）
5. 點擊「提交」按鈕將資料發送到 Google 表單

## 技術說明

- 使用 HTML5、CSS3 和 JavaScript 構建
- 使用 Google Forms API 儲存資料
- 響應式設計，適合各種設備使用
